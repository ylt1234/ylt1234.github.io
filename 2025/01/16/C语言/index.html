初级阶段

scanf(“ %c”, &j)

  • scanf(" %c", &j)和scanf("%c", &j)的区别是什么

  1. 当使用 scanf("%c", &j); 时，%c 格式说明符会读取下一个字符，无论该字符是什么，包括空格、换行符、制表符等空白字符。

    例如，如果在使用 scanf("%c", &j); 之前有一个 scanf("%d", &i); 操作，用户输入 10（实际是10\n ），scanf("%d", &i); 会读取 10，而 scanf("%c", &j); 会读取 \n

  2. 在 %c 前面添加一个空格，即 scanf(" %c", &j);，会指示 scanf 跳过输入缓冲区中的所有空白字符（包括空格、换行符、制表符等），然后读取下一个非空白字符。

    在上面例子中，就不会读取\n

  3. 除了%c，其他都能自己跳过空白字符，所以其他都不需要使用这一种格式

  4. 所以如果遇到连续几次读取输入的内容，要注意%c上面有没有读取其他内容

  5. gets也会读取空白字符，所以如果前面有其他读取的函数时，可以先使用scanf("%c",&c);来把空白字符先读走，再gets

函数的递归调用

需要找到两个目标：

  1. 找到一个函数，可以表示第n次调用和第n-1等次调用的关系
  2. 找到调用结束的状态

例如：假如有n个台阶，一次只能上1个台阶或2个台阶，请问走到第n个台阶有几种走法？为便于读者理解题意，这里举例说明如下：假如有3个台阶，那么总计就有3种走法：第一种为每次上1个台阶，上3次；第二种为先上2个台阶，再上1个台阶；第三种为先上1个台阶，再上2个台阶。输入为n，输出为走到第n个台阶有几种走法

这题的函数是 走到第n个台阶的走法数 = 走到第n-1个台阶的走法数 + 走到第n-2个台阶的走法数，所以step(n-1)+step(n-2)。这样一直往前推，step(n)=step(n-1)+step(n-2)=step(n-2)+step(n-3)+step(n-3)+step(n-4)，比如n取3的时候，step(3)=step(2)+step(1)，发现不能继续往下推了，因为step(0)没有意义，所以到n=2，或者n=1的时候就应该停止。当n=2时，走法数为2，当n=1时，走法数为1，所以当n==1||n==2时，直接return n

顺序表的插入删除查找

  • 插入流程：
  1. 判断i是否合法（是否超过顺序表的长度）
  2. 判断顺序表的内容是不是已满
  3. 把后面的元素依次往后移动。比如有一个长度为10的顺序表，我们想把新元素插在第7位，那就需要把原本第10位的，移动到第11位；把原本第9位的，移动到第10位；把原本第8位的，移动到第9位；把原本第7位的，移动到第8位；这就可以把第7位的位置空出来插入新元素了
  4. 顺序表长度要加1
  • 删除流程：
  1. 判断i是否合法（是否在1到顺序表最大长度之间）（注意在数据结构中，顺序表的第一位就称为第一位，而不是像数组那样称为第零位）
  2. 如果需要的话，用一个变量保存删除元素
  3. 把后面的元素依次向前移动，比如有一个长度为10的顺序表，我们想把第7位的元素删除掉，就要把原本的第8个元素之间覆盖掉原本的第7个元素；把原本的第9个元素之间覆盖掉原本的第8个元素；把原本的第10个元素之间覆盖掉原本的第9个元素；
  4. 顺序表长度要减1
  • 查找流程：
  1. 按顺序从顺序表第一个元素开始，和要查找的元素作比较。注意要放回的是数组下标+1，因为在数据结构中，顺序表的第一位就称为第一位，而不是像数组那样称为第零位

队列

在表的一端（尾部）进行插入，在表的另一端（头部）进行删除。特性是先进先出

初始化队列：Q.rear=Q.front=0

判断队列空：Q.rear=Q.front

判断队列满 ：(Q.rear+1)%MaxSize==Q.front

入队：Q.data=[Q.rear]=x;Q.rear=(Q.rear+1)%MaxSize

出队 ：x=Q.data[Q.front];Q.front=(Q.front+1)%MaxSize;

二叉树

使用辅助队列层次建树

输入abcdefghij一串字符，可以通过辅助队列——一个队列加上一个指针（用pcur来代表）来判断这个树的层级结构。

  1. 二叉树中写入a，为根结点直接写；辅助队列写入a，pcur指针指向a
  2. 二叉树中写入b，由于此时pcur指针指向a，而a此时还没有左叶子结点，所以b写到a的左叶子结点；辅助队列写入b，pcur指针不变
  3. 二叉树中写入c，由于此时pcur指针指向a，而a此时已经有左叶子结点，所以b写到a的右叶子结点；辅助队列写入b，pcur指针指向b（因为a已经有了右叶子结点）

二叉树的前序中序后序遍历

前序遍历：先打印自身，再打印左子树，再打印右子树
