PHP魔术空间&运行空间

有八个魔术常量它们的值随着它们在代码中的位置改变而改变。
<1>LINE
文件中的当前行号。
<2>FILE
文件的完整路径和文件名。如果用在被包含文件中，则返回被包含的文件名。
<3>DIR
文件所在的目录。如果用在被包括文件中，则返回被包括的文件所在的目录。
<4>FUNCTION
本常量返回该函数被定义时的名字（区分大小写）。
<5>METHOD
类的方法名（PHP 5.0.0 新加）。返回该方法被定义时的名字（区分大小写）
<6>NAMESPACE
当前命名空间的名称（区分大小写）。
<7>CLASS
本常量返回该类被定义时的名字（区分大小写）
<8>TRAIT
同class，但是如果你要让这个类被中的方法被继承使用，那么就需要使用trait
***<1>-<6>较好理解，举下<7><8>的例子
Eg1：class test {
function _print() {
echo ‘类名为：’ . CLASS . “
“;
echo ‘函数名为：’ . FUNCTION ;
}
$t = new test();
$t->_print();
?>
运行的结果为：
类名为：test
函数名为：_print
Eg2：
class Base {
public function sayHello() {
echo ‘Hello ‘;
}
trait SayWorld {
public function sayHello() {
parent::sayHello();
echo ‘World!’;
}
}
class MyHelloWorld extends Base {
use SayWorld;
}
$o = new MyHelloWorld();
$o->sayHello();
运行的结果为：
Hello World!

PHP判断与循环&函数

*判断与循环
If..else、switch、while循环、do..while循环、for 循环 与c语言相仿。
单独讲讲foreach 循环，foreach 循环用于遍历数组。
语法：
1、foreach ($array as $value)
{
要执行代码;
}
每进行一次循环，当前数组元素的值就会被赋值给 $value 变量（数组指针会逐一地移动）。
Eg：
$x=array(“Google”,”Runoob”,”Taobao”);
foreach ($x as $value)
{
echo $value . PHP_EOL;
}
运行的结果为：
Google
Runoob
Taobao
2、foreach ($array as $key => $value)
{
要执行代码;
}
每一次循环，当前数组元素的键与值就都会被赋值给 $key 和 $value 变量（数字指针会逐一地移动）
Eg：
$x=array(1=>”Google”, 2=>”Runoob”, 3=>”Taobao”);
foreach ($x as $key => $value)
{
echo “key 为 “ . $key . “，对应的 value 为 “. $value . PHP_EOL;
}
运行的结果为：
key 为 1，对应的 value 为 Google
key 为 2，对应的 value 为 Runoob
key 为 3，对应的 value 为 Taobao

*函数
1、PHP 函数准则：
<1>函数的名称应该提示出它的功能
<2>函数名称以字母或下划线开头（不能以数字开头）
2、创建 PHP 函数：函数是通过调用函数来执行的。
3、语法：function functionName()
{
// 要执行的代码
}
Eg：
function writeName()
{
echo “Kai Jim Refsnes”;
}
echo “My name is “;
writeName();
运行的结果为：
My name is Kai Jim Refsnes
Ps：我对function理解的意思是 把writeName()等价于echo “Kai Jim Refsnes”;
4、 添加参数
参数就在函数名称后面的一个括号内指定。
Eg：function writeName($fname,$punctuation)
{
echo $fname . “ Refsnes” . $punctuation . “
“;
}
echo “My name is “;
writeName(“Kai Jim”,”.”);
运行的结果为：
My name is Kai Jim Refsnes.
5、返回值
返回值，就是你运用一个函数，函数结束后，有一个结果，如果你需要那个结果，你就需要返回值
Eg：
function add($x,$y)
{
$total=$x+$y;
return $total;
}
echo “1 + 16 = “ . add(1,16);
运行的结果为：
1 + 16 = 17

$_SERVER 变量中的重要元素

下表列出了所有 $_SERVER 变量中的重要元素:
*$SERVER[‘PHP_SELF’]
当前执行脚本的文件名，与 document root 有关。例如，在地址为 http://example.com/test.php/foo.bar 的脚本中使用 $SERVER[‘PHP_SELF’] 将得到 /test.php/foo.bar。__FILE 常量包含当前(例如包含)文件的完整路径和文件名。 从 PHP 4.3.0 版本开始，如果 PHP 以命令行模式运行，这个变量将包含脚本名。之前的版本该变量不可用。
*$_SERVER[‘GATEWAY_INTERFACE’]
服务器使用的 CGI 规范的版本；例如，”CGI/1.1”。
*$_SERVER[‘SERVER_ADDR’]
当前运行脚本所在的服务器的 IP 地址。
*$_SERVER[‘SERVER_NAME’]
当前运行脚本所在的服务器的主机名。如果脚本运行于虚拟主机中，该名称是由那个虚拟主机所设置的值决定。(如: www.runoob.com)
*$_SERVER[‘SERVER_SOFTWARE’]
服务器标识字符串，在响应请求时的头信息中给出。 (如：Apache/2.2.24)
*$_SERVER[‘SERVER_PROTOCOL’]
请求页面时通信协议的名称和版本。例如，”HTTP/1.0”。
*$_SERVER[‘REQUEST_METHOD’]
访问页面使用的请求方法；例如，”GET”, “HEAD”，”POST”，”PUT”。
*$_SERVER[‘REQUEST_TIME’]
请求开始时的时间戳。从 PHP 5.1.0 起可用。 (如：1377687496)
*$_SERVER[‘QUERY_STRING’]
query string（查询字符串），如果有的话，通过它进行页面访问。
*$_SERVER[‘HTTP_ACCEPT’]
当前请求头中 Accept: 项的内容，如果存在的话。
$_SERVER[‘HTTP_ACCEPT_CHARSET’]
当前请求头中 Accept-Charset: 项的内容，如果存在的话。例如：”iso-8859-1,,utf-8”。
*$_SERVER[‘HTTP_HOST’]
当前请求头中 Host: 项的内容，如果存在的话。
*$_SERVER[‘HTTP_REFERER’]
引导用户代理到当前页的前一页的地址（如果存在）。由 user agent 设置决定。并不是所有的用户代理都会设置该项，有的还提供了修改 HTTP_REFERER 的功能。简言之，该值并不可信。)
*$_SERVER[‘HTTPS’]
如果脚本是通过 HTTPS 协议被访问，则被设为一个非空的值。
*$_SERVER[‘REMOTE_ADDR’]
浏览当前页面的用户的 IP 地址。
*$_SERVER[‘REMOTE_HOST’]
浏览当前页面的用户的主机名。DNS 反向解析不依赖于用户的 REMOTE_ADDR。
*$_SERVER[‘REMOTE_PORT’]
用户机器上连接到 Web 服务器所使用的端口号。
*$_SERVER[‘SCRIPT_FILENAME’]
当前执行脚本的绝对路径。
*$_SERVER[‘SERVER_ADMIN’]
该值指明了 Apache 服务器配置文件中的 SERVER_ADMIN 参数。如果脚本运行在一个虚拟主机上，则该值是那个虚拟主机的值。(如：someone@runoob.com)
*$_SERVER[‘SERVER_PORT’]
Web 服务器使用的端口。默认值为 “80”。如果使用 SSL 安全连接，则这个值为用户设置的 HTTP 端口。
*$_SERVER[‘SERVER_SIGNATURE’]
包含了服务器版本和虚拟主机名的字符串。
*$SERVER[‘PATH_TRANSLATED’]
当前脚本所在文件系统（非文档根目录）的基本路径。这是在服务器进行虚拟到真实路径的映像后的结果。
*$SERVER[‘SCRIPT_NAME’]
包含当前脚本的路径。这在页面需要指向自己时非常有用。__FILE 常量包含当前脚本(例如包含文件)的完整路径和文件名。
$_SERVER[‘SCRIPT_URI’]
URI 用来指定要访问的页面。例如 “/index.html”。
Ps：无号，只是为了方便查阅。

PHP 超级全局变量列表:
<1>$GLOBALS

1、$GLOBALS
$GLOBALS 是一个包含了全部变量的全局组合数组。

addition();

100

PHP数组

***数组在之前已经简单提过，这里将更详细的介绍
1、是什么：数组是一个能在单个变量中存储多个值的特殊变量。

2、 创建数组：array() 函数用于创建数组：array();
这里有两种创建数值数组的方法：
<1>自动分配 ID 键（ID 键总是从 0 开始）：
Eg
$cars=array(“Volvo”,”BMW”,”Toyota”);
<2>人工分配 ID 键：
Eg:
$cars[0]=”Volvo”;
$cars[1]=”BMW”;
$cars[2]=”Toyota”;

3、获取数组的长度 - count() 函数
Eg：
$cars=array(“Volvo”,”BMW”,”Toyota”);
echo count($cars);
运行的结果为：
3

4、遍历数值数组：遍历并打印数值数组中的所有值，可以使用 for 循环
Eg：
$cars=array(“Volvo”,”BMW”,”Toyota”);
$arrlength=count($cars);
for($x=0;$x<$arrlength;$x++)
{
echo $cars[$x];
echo “
“;
}
运行的结果为：
Volvo
BMW
Toyota

5、PHP 关联数组：关联数组是使用您分配给数组的指定的键的数组。
Eg：
$age=array(“Peter”=>”35”,”Ben”=>”37”,”Joe”=>”43”);
or:
$age[‘Peter’]=”35”;
$age[‘Ben’]=”37”;
$age[‘Joe’]=”43”;
6、遍历关联数组：遍历并打印关联数组中的所有值，可以使用 foreach 循环
Eg：
$age=array(“Peter”=>”35”,”Ben”=>”37”,”Joe”=>”43”);
foreach($age as $x=>$x_value)
{
echo “Key=” . $x . “, Value=” . $x_value;
echo “
“;
}
运行的结果为：
Key=Peter, Value=35
Key=Ben, Value=37
Key=Joe, Value=43
Ps：foreach函数用法过几篇再说明
7、数组排序函数
<1>sort() - 对数组进行升序排列
<2>rsort() - 对数组进行降序排列
<3>asort() - 根据关联数组的值，对数组进行升序排列
<4>ksort() - 根据关联数组的键，对数组进行升序排列
<5>arsort() - 根据关联数组的值，对数组进行降序排列
<6>krsort() - 根据关联数组的键，对数组进行降序排列
每个函数相差不大，简单拿<1><3>举两个例子
Eg1：
$cars=array(“Volvo”,”BMW”,”Toyota”);
sort($cars);
print_r($cars);
运行的结果为：
Array ( [0] => BMW [1] => Toyota [2] => Volvo )
Eg2:
$age=array(“Peter”=>”35”,”Ben”=>”37”,”Joe”=>”43”);
asort($age);
运行的结果为：
Array ( [Peter] => 35 [Ben] => 37 [Joe] => 43 )

PHP运算符

和c语言有很多共同之处，我就不在这里赘述了，讲讲不同的。

1、~x 取反
x 取反，按二进制位进行”取反”运算。
Eg：
~1=-2;
~0=-1;

2、整除运算符 intdiv()
eg：
var_dump(intdiv(10, 3));
运行的结果为：
int(3)

3、x != y或x <> y 不等于

4、x xor y 异或 如果 x 和 y 有且仅有一个为 true，则返回 true
Eg：
x=6
y=3
(x==6 xor y==3) 返回 false

5、三元运算符
<1>语法格式：(expr1) ? (expr2) : (expr3)
对 expr1 求值为 TRUE 时的值为 expr2，在 expr1 求值为 FALSE 时的值为 expr3。
<2> 语法格式：expr1 ?: expr3
在 expr1 求值为 TRUE 时返回 expr1，否则返回 expr3。
<3>语法格式：expr1 ?? expr3
在 expr1 求值为 TRUE 时返回 expr1，否则返回 expr3。

6、组合比较符
符号为 <=>。组合比较运算符可以轻松实现两个变量的比较，当然不仅限于数值类数据的比较。
语法格式如下：
$c = $a <=> $b;
解析如下：
如果 $a > $b, 则 $c 的值为 1。
如果 $a == $b, 则 $c 的值为 0。
如果 $a < $b, 则 $c 的值为 -1
Eg：echo 1 <=> 1; // 0
echo 1 <=> 2; // -1
echo 2 <=> 1; // 1

PHP字符串变量

字符串变量的概念在数据类型那篇已经介绍过了，这里便不再重复。

1、 PHP 并置运算符
并置运算符 (.) 用于把两个字符串值连接起来。
Eg：
$txt1=”Hello world!”;
$txt2=”What a nice day!”;
echo $txt1 . “ “ . $txt2;
输出的结果为：
Hello world! What a nice day!
***在上面的代码中，我们已经使用了两次并置运算符。这是由于我们需要在两个字符串之间插入一个空格。

2、 PHP strlen() 函数
strlen() 函数返回字符串的长度（字节数）。
Eg:
echo strlen(“Hello world!”);
输出的结果为：
12

3、 PHP strpos() 函数
strpos() 函数用于在字符串内查找一个字符或一段指定的文本。如果在字符串中找到匹配，该函数会返回第一个匹配的字符位置。如果未找到匹配，则返回 FALSE。
Eg：
echo strpos(“Hello world!”,”world”);
运行的结果为：
6
***在上面的实例中，字符串 “world” 的位置是 6。之所以是 6 而不是 7 的原因是，字符串中第一个字符的位置是 0，而不是 1。
