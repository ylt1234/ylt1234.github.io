PHP变量

与代数类似，可以给 PHP 变量赋予某个值（x=5）或者表达式（z=x+y）。变量可以是很短的名称（如 x 和 y）或者更具描述性的名称（如 age、carname、totalvolume）。
1、 变量以 $ 符号开始，后面跟着变量的名称

2、 PHP 没有声明变量的命令。变量在第一次赋值给它的时候被创建：$txt=”Hello world!”
***当赋一个文本值给变量时，在文本值两侧加上引号。

3、 PHP 是一门弱类型语言，不必向 PHP 声明该变量的数据类型。

4、 局部和全局作用域：变量在函数外定义，无法在函数内使用；变量在函数中定义，无法在函数外使用。
Eg：
$x=5; // 全局变量
function myTest()
{
$y=10; // 局部变量
echo “

测试函数内变量:

“;
echo “变量 x 为: $x”; (x无输出)
echo “
“;
echo “变量 y 为: $y”; （y=10）
} 

myTest();
echo "<p>测试函数外变量:<p>";
echo "变量 x 为: $x";    （x=5）
echo "<br>";
echo "变量 y 为: $y";     （y无输出）

6、PHP 变量作用域：脚本中变量可被引用/使用的部分。
PHP 有四种不同的变量作用域：local、global、static、parameter
<1>global:用于函数内访问全局变量。
Eg:
$x=5;
$y=10;
function myTest()
{
global $x,$y;
$y=$x+$ y;
}
myTest();
echo $y; // 输出 15
<2> static：当一个函数完成时，它的所有变量通常都会被删除。然而，有时候希望某个局部变量不要被删除。要做到这一点，在第一次声明变量时使用 static 关键字。
Eg：
function myTest()
{
static $x=0;
echo $x;
$x++;
echo PHP_EOL; // 换行符
}

体验CTF——攻防世界web基础

1、查看源代码–ctrl+u
2、用GET方式提交一个名为a,值为1的变量–在网址上加 /?a=1(?为中文)
3、以POST方式随便提交一个名为b,值为2的变量—在hackbar中加b=2
4、robots协议也称爬虫协议，告诉搜索引擎哪些页面可以抓取,哪些页面不能抓取。—在网址后加/robots.txt来访问
5、index.php的备份文件—在网址后加/index.php.bak
6、查看cookie— f12-网络-刷新
7、不能按的按钮： —disable禁用了input，删去disable
8、摩斯密码：11 111 010 000 0 1010 111 100 0 00 000 000 111 00 10 1 0 010 0 000 1 00 10 110
（看构成仅有0，1，可能为摩斯密码，或者转为ascii，又看到0，1分布有空格隔开，且没有规律，判定为摩斯密码）将1-为将0为.，或者互换，空格用/ 去在线转换网址查看
9、 一句话木马，使用蚁剑，蚁剑的作用简单来说就是连接shell，shell相当于命令行和用户之间的“代理”。
