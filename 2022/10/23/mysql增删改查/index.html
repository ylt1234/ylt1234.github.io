mysql增删改查

Sun October 23, 2022 10:40 PM

*库操作
1、创建数据库:create database +（数据库名）；
2、显示数据库:show databases;
3、删除数据库：drop database shujuku;
*增删改查的语句命令为：
操作 命令
增 insert
删 delete
改 update
查 select或者show
Eg：增删改查实例
1、准备表
create table employee(
id int,
name varchar(40),
sex varchar(4),
birthday date,
entry_date date,
salary decimal(8,2),
resum text
);
2、插入数据
insert into employee(id,name,sex,birthday,entry_date,salary,resum)
values(1,’zhangsan’,’male’,’1993-03-04’,’2016-11-10’,’1000’,’i am a developer’);
3、 修改表数据
将姓名为‘zhangsan’的员工薪水修改为3000元：
update employee set salary=3000 where name=’zhangsan’;
4、 删除表数据
<1>删除表中名称为‘zhangsan’的记录：
delete from employee where job=’zhangsan’;
<2>使用truncate删除表中记录：
truncate table employee;
5、查询表中所有学生的姓名和对应的英语成绩：
select name,english from student;
